# 🛒 Sistema de Controle de Estoque

Bem-vindo ao  **Sistema de Controle de Estoque** ! Este projeto foi desenvolvido para ajudar a gerenciar o estoque de produtos de forma simples e eficiente, utilizando Python, SQLite para banco de dados e o Pandas para exibição de dados em formato de tabela no Jupyter Notebook. Além disso, o projeto conta com uma interface gráfica para facilitar o uso, construída com o  **Tkinter** .

## 📋 Funcionalidades

O sistema oferece as seguintes funcionalidades:

* **Adicionar produtos** : Registre novos produtos com nome, quantidade e preço.
* **Listar produtos** : Veja uma lista completa de todos os produtos no estoque.
* **Atualizar produtos** : Altere os dados de produtos já cadastrados (nome, quantidade, preço).
* **Remover produtos** : Exclua produtos do sistema de estoque.
* **Interface gráfica (GUI)** : Use uma interface amigável para gerenciar o estoque.
* **Visualização em Tabela** : Exiba os produtos em formato de tabela no Jupyter Notebook.

## 🛠️ Tecnologias Utilizadas

* **Python** 🐍: Linguagem principal do projeto.
* **SQLite** 🗄️: Banco de dados para armazenar as informações dos produtos.
* **Tkinter** 🖥️: Biblioteca para criar a interface gráfica do usuário.
* **Pandas** 🧮: Biblioteca para exibir os dados em formato de tabela.
* **Jupyter Notebook** 📓: Ambiente interativo para visualização e desenvolvimento.

## 📚 Documentação das Funções

### `adicionar_produto(nome, quantidade, preco)`

Adiciona um novo produto ao banco de dados.

### `listar_produtos()`

Retorna uma lista de todos os produtos cadastrados.

### `atualizar_produto(id_produto, nome, quantidade, preco)`

Atualiza os detalhes de um produto existente.

### `remover_produto(id_produto)`

Remove um produto do banco de dados.

### `exibir_estoque_tabela()`

Exibe os produtos cadastrados em formato de tabela no Jupyter Notebook.
